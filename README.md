# Insurance Contribution System

This project is a .NET Core backend API developed as part of a technical assessment.

## Tech Stack
- .NET Core 8/9
- C#
- Entity Framework Core
- SQL Server
- Swagger (OpenAPI)

## Features
- Create insurance contributions
- Validate policies and transactions
- Aggregated contribution summary by policy
- Clean architecture and layered design

## APIs
- POST /api/contributions
- GET /api/policies/{policyNumber}/contributions

## Setup
1. Update connection string in `appsettings.json`
2. Run database scripts / migrations
3. Run the API
4. Access Swagger at `/swagger`
